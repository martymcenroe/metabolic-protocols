# Metabolic Protocols

A structured repository for bio-hacking, zero-carb nutrition, and metabolic data management.

## Project Goal

To engineer a reproducible system for lowering A1C, reducing visceral fat, and optimizing athletic recovery through strict Zero-Carb/LCHF protocols.

## Usage

This project uses **Poetry** for dependency management.

```
poetry install
```
